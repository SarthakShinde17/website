# Sarthak Shinde - Portfolio Website

Welcome to the repository for my personal portfolio website! I am a **Data Analyst & AI Engineer** specializing in Generative AI, Data Visualization, and automated workflows.

This website serves as an interactive showcase of my experience, projects, skills, and professional journey.

## Features

- **Modern & Interactive UI**: Built with React and structured for a seamless user experience.
- **3D Hero Section**: Features interactive 3D elements powered by `three.js`, `@react-three/fiber`, and `@react-three/drei`.
- **Smooth Animations**: Transitions and element animations handled by `framer-motion`.
- **Fully Responsive**: Designed to look great on devices of all sizes.
- **Sections Included**: 
  - **About / Hero**: Introduction and 3D visual.
  - **Experience**: My professional timeline and roles.
  - **Projects**: Highlighted past work and technical achievements.
  - **Skills & Education**: Overview of my technical stack and academic background.
  - **Contact**: Reach out to me directly.

## Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [Three.js](https://threejs.org/) & [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Custom Vanilla CSS

## Getting Started

To run this project locally, simply follow these steps:

1. **Clone the repository** (or download the source code):
   ```bash
   git clone <repository-url>
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```


## Scripts

- `npm run dev` - Starts the Vite development server.
- `npm run build` - Builds the app for production.
- `npm run preview` - Locally previews the production build.
- `npm run lint` - Runs ESLint.
